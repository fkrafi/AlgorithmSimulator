Depth First Search

The principle of the Depth Frist Search algorithm is to go forward (in depth) while there is such possibility, otherwise to backtrack.

Initially all vertices are white (unvisited). DFS starts in arbitrary vertex and runs as follows:

  1. Mark vertex u as gray (visited).
  2. For each edge (u, v), where u is white, run depth-first search for u recursively.
  3. Mark vertex u as black and backtrack to the parent.

Example


Pseudo Code

DFS(G){
    for each u in V do
    color[u] = WHITE
    pi[u] = NILL
    time=0;
    for each u in V do
        if color[u] == white
            DFSVisit(u)
}

DFSVisit(u) {
    color[u] = GRAY;
    d[u] = ++time;
    for each v in Adj(u) do
        if color[v] == WHITE
            pi[v] = u
            DFSVisit(v)
    color[u] = black;
    f[u] = ++time;
}

Time Complexity

The time complexity can be expressed as O(|V|+|E|).
