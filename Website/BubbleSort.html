Bubble Sort

The bubble sort is generally considered to be the simplest sorting algorithm. Because of its simplicity and ease of visualizatio The bubble sort works by passing sequentially over a list, comparing each value to the one immediately after it. If the first value is greater than the second, their positions are switched. Over a number of passes, at most equal to the number of elements in the list, all of the values drift into their correct positions.

Pseudo Code

for i=1:length(A)
    for j=length(A):i+1
        if A[j] < A[j-1], 
            swap(A[j], A[j-1])

Time Complexity

Time complexity of bubble sort is O(n2).
