Merge Sort

The sorting algorithm Mergesort produces a sorted sequence by sorting its two halves and merging them. Mergesort works as follows :

  1. If the input sequence has fewer than two elements, return.
  2. Partition the input sequence into two halves.
  3. Sort the two subsequences using the same algorithm.
  4. Merge the two sorted subsequences to form the output sequence.

Example

Pseudo Code

Merge(A, p, q, r){
    n1 = q-p+1
    n2 = r-q
    for i=1:n1
        L[i] = A[p+i-1]
    for j=1:n2
        R[j] = A[q+j]
    L[n1+1] = INF
    R[n2+1] = INF
    i = 1
    j = 1
    for j=p:r
        if L[i] < R[j]
            A[k] = L[i]
            i = i + 1
        else
            A[k] = R[j]
            j = j + 1
}

MergeSort(A, p, r){
    q = (p+r)/2
    MergeSort(A, p, q)
    MergeSort(A, q+1, r)
    Merge(A, p, q, r)
}

Time Complexity

Time complexity of mergesort is O(n log n).
